# Day 2 — Building a UI Component Library (Buttons, Cards, Inputs)

## Objective
Build reusable UI components using TailwindCSS and organize them properly.

##  Topics Covered
- Atomic design principles
- Reusable UI components
- Tailwind variants
- Props design

##  Tasks / Exercises
Create the following components:
- Button
- Input
- Card
- Badge
- Modal

##  Folder Structure
/components/ui/Button.jsx  
/components/ui/Card.jsx  
/components/ui/Input.jsx  
/components/ui/Badge.jsx  
/components/ui/Modal.jsx  

## ▶ How to Run

npm install
npm run dev


## Components Built
- `<Button variant="primary" />`
- `<Input />`
- `<Card />`
- `<Badge />`
- `<Modal />`



##  Notes / Learnings
- Use props to make components flexible.
- Component reuse improves consistency and speed.
